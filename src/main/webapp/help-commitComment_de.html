Dieser Wert wird als Vorlage für die Subversion commit message verwendet, und als Standardwert weitergegeben, wenn das Plugin aktiviert ist. Die folgenden Variablen können im groovy string format verwendet werden.
env
Map returned by hudson.model.AbstractBuild.getEnvVar().
sys
Properties returned by System.getProperties().
Zum Beispiel, kann BUILD_ID in der Subversion commit message verwendet werden. 
        Commit by Jenkins svn-commit-plugin. Build: ${env['BUILD_ID']}
